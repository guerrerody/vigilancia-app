# vigilancia-app
Proyecto de Sistema de Vigilancia para Laboratorio I de la UCLA.

Para compilar sin necesidad de tener Maven Instalado

##### Para Linux:
```bash
./mvnw clean package
```

##### Para Windows:
```bash
mvnw.cmd clean package
```

##### Para Ejecutar:
```bash
java -jar target/vigilancia-app-jar-with-dependencies.jar
```
